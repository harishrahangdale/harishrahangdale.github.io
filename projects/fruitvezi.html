Online Fruit & Vegetable Selling Platform

  • Tech Stack: PHP, JavaScript, Ajax, HTML, CSS, Bootstrap, PrestaShop API
  • Website URL: Link

Built on top of PrestaShop opensource Ecommerce framework.

User can register/login to the website, search for different categories/brands of fruitvezis, view detailed information of product, search product based on filters, add/remove from cart, order and pay for products.

Admin can manage the complete inventory, order details, customer details, logistics and website frontend through powerful backoffice-dashboard.
